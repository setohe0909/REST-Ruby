# Crear una API JSON en Ruby on Rails 4

Se creara un servicio utilizando Ruby on Rails en su cuarta versión; donde, podremos transmitir y obtener información en formato JSON.

## Servicios RESTful

Los Servicios RESTful son servicios muy particulares donde solo se definen 4 métodos para poder realizar todas nuestras tareas, los mismos son GET para consultar datos, POST para enviar datos, PUT para actualizar datos y DELETE para eliminar datos.
